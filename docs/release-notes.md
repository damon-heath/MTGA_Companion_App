# Arena Companion v1.0.0 Release Notes (Draft)

## Highlights
- Local-first ingestion and analytics pipeline.
- Match, deck, timeline, and observed-opponent tracking.
- CSV and JSON exports.

## Known Limits
- Collection module is de-scoped from 1.0.0 after spike no-go decision.
- Windows-only support in 1.0.0.
- Packaging mode for 1.0.0 is one-folder (see `docs/packaging-mode-decision.md`).
