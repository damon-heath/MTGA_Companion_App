# Release Checklist

## Functional Gates
- [ ] Replay + live ingest verified on current MTGA logs.
- [ ] Parser outputs validated against fixture corpus.
- [ ] Opponent observed cards and timeline views verified.
- [ ] CSV/JSON exports validated in spreadsheet and JSON consumers.
- [ ] Offline runtime guard test passing.

## Packaging Gates
- [ ] PyInstaller one-folder build completed.
- [ ] Installer built and validated on clean machine.
- [ ] Checksums generated for release artifacts.

## Legal / Distribution
- [ ] Mark product as unofficial fan-made software.
- [ ] Do not ship Wizards-owned logos/assets without permission.
- [ ] Include Wizards Fan Content Policy disclaimer in release notes.
